# NexaDesk Brand Identity

## Brand Name
**NexaDesk** — "Nexa" (next/nexus) + "Desk" (desktop workbench)

## Tagline
*下一代多智能体桌面工作台 · Next-Gen Multi-Agent Desktop Workbench*

## Logo
- Primary mark: Hexagonal icon with "ND" monogram
- Color: Deep green (#1f6b50) on dark backgrounds
- Format: SVG / PNG

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| Primary | #1f6b50 | Brand color, buttons, links |
| Primary Hover | #2e8b68 | Interactive states |
| Surface | #1a1d27 | Panel backgrounds |
| Background | #13151d | App background |
| Text Primary | #e4e5e9 | Primary text |
| Text Secondary | #9ca3af | Secondary/muted text |
| Danger | #ef4444 | Errors, destructive actions |
| Warning | #f59e0b | Warnings |

## Typography
- Font: system-ui, -apple-system, sans-serif
- Monospace: JetBrains Mono / Fira Code for code blocks

## Voice & Tone
- Professional but approachable
- Technical but clear
- Chinese primary, English secondary

## Core Values
1. **Multi-Agent by Design** — Not a chatbot, a multi-agent workbench
2. **Local-First** — Privacy-first, offline-capable
3. **Extensible** — MCP protocol, plugin architecture
4. **Developer Friendly** — CLI tools, Git integration, code-aware
